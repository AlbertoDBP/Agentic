```bash

cd /path/to/Agentic

# 2. Configure git (first time only)
git config user.name "Your Name"

# 3. Add all files

# 4. Create initial commit
git commit -m "Initial commit: Platform Documentation Orchestrator skill"

# 5. Add remote (replace with your actual repo URL)
```
